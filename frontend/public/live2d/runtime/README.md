# Live2D Runtime Directory

Place the official Cubism Core runtime file here after downloading the Live2D
Cubism SDK for Web:

```text
live2dcubismcore.min.js
```

Default frontend runtime URL:

```text
VITE_LIVE2D_CORE_URL=/live/live2d/runtime/live2dcubismcore.min.js
```

The Cubism Core file is not bundled in this repository because it must be
obtained from the official Live2D SDK download under its license terms.
