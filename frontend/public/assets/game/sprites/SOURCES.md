# Game Sprite Sources

This visual pack combines Kenney CC0 public mature assets, AI-generated Hoshia/gameplay assets, and local post-processing for atlas-ready sprites.

## Kenney CC0 public mature assets

- Kenney Tiny Dungeon - Creative Commons CC0. Source: https://kenney.nl/assets/tiny-dungeon
- Kenney Top-down Shooter - Creative Commons CC0. Source: https://kenney.nl/assets/top-down-shooter
- Kenney Particle Pack - Creative Commons CC0. Source: https://kenney.nl/assets/particle-pack

Kenney attribution is not required for CC0 assets, but is included here for clarity. No Kenney logos are included.

## AI-generated Hoshia/gameplay assets

- Hoshia player sprite, class portraits, class icons, weapon concepts, enemy/boss remaps, and selected neon combat effects are AI-generated or AI-assisted for Hoshia Live Room, then curated by the project.
- These assets follow the project theme: white-haired cat-ear idol, blue/white stage outfit, purple eyes, headset, key/paw motifs, cyber-radio weapons, and pixel-survivor combat effects.

## Local post-processing and atlas packing

- Final committed sprites may be cropped, resized, recolored, transparency-cleaned, renamed, composited, previewed, and packed into atlas files for runtime use.
- Post-processing outputs only public game art files and metadata. Temporary source caches, raw external packs, prompts, credentials, private links, and deployment details are not included.

## Excluded sources

Cainos, itch.io non-CC0 packages, paid packs, fan art, game rips, private assets, credentials, and private deployment links are not included.
