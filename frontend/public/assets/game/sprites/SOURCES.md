# Game Sprite Sources

This visual pack combines mature public-domain source assets with project-specific Hoshia pixel art.

## Public mature assets used

- Kenney Tiny Dungeon - Creative Commons CC0. Source: https://kenney.nl/assets/tiny-dungeon
- Kenney Top-down Shooter - Creative Commons CC0. Source: https://kenney.nl/assets/top-down-shooter
- Kenney Particle Pack - Creative Commons CC0. Source: https://kenney.nl/assets/particle-pack

Kenney attribution is not required for CC0 assets, but is included here for clarity. No Kenney logos are included.

## Project-specific additions

- Hoshia player sprite, class portraits, enemy remaps, boss remaps, icons, and neon combat effect names are composed for Hoshia Live Room from the project brief and/or procedural pixel drawing.
- Cainos, itch.io non-CC0 packages, paid packs, fan art, game rips, private assets, credentials, and private deployment links are not included.
